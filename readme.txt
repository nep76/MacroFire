MacroFire - BETA2 Version


はじめに
-------------------------------------------------
  このプログラムは、PSPのカスタムファームウェア上で動作するプラグインです。
  通常のSony純正ファームウェア上では動作しません。

  BETAバージョンと銘打っているだけあって、
  全ての項目が実装されているわけではありません。
  マクロメニューの一部はまだ実装されていません。

  お決まりですが、何が起きても自己責任です。
  突然フリーズして長時間のプレイが水の泡になったりしても何もしてあげられません。


概要
-------------------------------------------------
  任意のボタンの連打設定や、
  一定の操作を自動で行わせるマクロ機能があります。

  またRapidfireの設定では連打の他に、押しっぱなし状態を設定することもできるので、
  RPGなどでは、輪ゴムや重石を使わなくても自動戦闘の類が可能です。

  マクロ機能は、特定の動作を記録させ、それを繰り返し実行することができます。


動作環境
-------------------------------------------------
  Custum Firmware 3.90 M33-3

  これ以外のファームウェアでは確認していません。
  ですが、3.71以下のファームウェアでは動作しないと思われます。

  新型のPSP-2000は持っていないので動作状況はわかりませんし、
  拡張されたRAMも使っていません。


インストール
-------------------------------------------------
  その他のプラグイン同様に、以下のファイルを
  ms0:/sepluginsフォルダにコピーします。

    macrofire.prx

  続いて、ms0:/seplugins内の game.txt に
  それぞれ以下のパスを追加します。
  これらのファイルが存在しない場合は作成してください。

    ms0:/seplugins/macrofire.prx

  書き込んだらリカバリーモードへ行き、Pluginsの項目から
  必要な場所のプラグインをEnabledにします。


使い方
-------------------------------------------------
  ゲームを起動したら、音量調節に使う二つのボタン(+と-のキー)を押すことでメニューを起動します。
  メニューでの基本的なキー操作は以下の通りです。

    ○ボタン   : 決定、及びオプション項目の切り替え
    ×ボタン   : 戻る
    STARTボタン: メニューをすぐに終了する

  これは一般的な操作であって、その他の機能別の項目によっては、
  さらに別の操作がある場合もあります。
  この場合は、画面下にキー毎の操作方法が書かれている場合があります。


機能
-------------------------------------------------
  MacroFire Engine
    このプラグインを作動させるかを切り替えます。
    ボタン連打機能やマクロ機能など、全ての機能に影響します。
    ゲーム起動直後はOFFになっているので、使用する場合は○ボタンを押してONに切り替えます。

    技術的な話になりますが、この項目がONにされた時に以下のAPIがフックされます。
      sceCtrlPeekBufferPositive
      sceCtrlPeekBufferNegative
      sceCtrlReadBufferPositive
      sceCtrlReadBufferNegative
      sceCtrlPeekLatch
      sceCtrlReadLatch
    ですが、今のところsceCtrl*Latch系のフック関数が正常に動作していません。
    これによって、ゲームそのものが正常に動作しなくなる場合や、
    特定のゲームモード、あるいはミニゲームだけ動作しないと言った可能性が考えられます。

    このような場合、この不完全なフック関数を使用していることが考えられますので、
    OFFに切り替えて様子を見てください。
    OFFにすることによってフックしているAPIを本来の物でリストアします。

    それ以外に単純にプラグインによってメモリ不足になることもあるでしょう。
    こうなるとゲーム起動でフリーズしたり、プレイ中に突然フリーズしたりすることが考えられます。
    これはどうしようもないのでカスタムファームウェアのリカバリメニューからプラグインを無効にしてください。


  Rapidfire settings
    ボタンの連打設定を行います。
    設定可能なボタンは以下のボタンです。

      ○ボタン      (Circle)
      ×ボタン      (Cross)
      □ボタン      (Square)
      △ボタン      (Triangle)
      十字キー上    (Up)
      十字キー右    (Right)
      十字キー下    (Down)
      十字キー左    (Left)
      Lボタン       (L-trig)
      Rボタン       (R-trig)
      セレクトボタン(SELECT)
      スタートボタン(START)

    それぞれのボタンに対して以下のモードを設定できます。

      NORMAL    : 通常の操作モードです。
      RAPID     : そのボタンを押している間だけ連打を行います。
      AUTO-RAPID: そのボタンを押しているかどうかに関わらず、常に連打します。
      AUTO-HOLD : そのボタンを常に押しっぱなし状態にします。


  Macro settings
    任意のキー操作を記録して、同じ操作を再実行可能にします。
    以下はマクロメニューの解説です。

    Load from MS (未実装です)
      メモリースティックに保存しされているマクロファイルを
      メインメモリに読み込みます。

    Save to MS (未実装です)
      メインメモリ上のマクロをメモリースティックに保存します。

    Edit macro
      メインメモリ上のマクロをを編集します。

    Clear macro
      メインメモリ上のマクロを破棄し、メモリを解放します。

    Start recording
      マクロのリアルタイム記録を開始します。
      この項目を選択すると、3秒後にゲーム画面に自動的に戻ります。
      ゲーム画面に戻った後、後述のStop recordingを選択するまでの
      ゲーム上での操作をすべて記録します。

      記録できる行動数に限界は設定していませんが、
      メモリの限界になると一応それ以上記録しないようにしているつもりです。
      ですが、いっぱいまで記録するのは危険ですのであまりに長いマクロは作らないようにしてください。
      また、使い終わったマクロはなるべくこまめにClearした方がいいかもしれません。
      Clearすることによってメモリが解放されます。

    Stop recording
      マクロのリアルタイム記録を終了します。

    Run once
      メインメモリ上のマクロを一度だけ実行します。

    Run infinity
      メインメモリ上のマクロを、後述のStop macroで停止するまで、
      延々とループして実行し続けます。

    Stop macro
      Run once、Run infinityによって実行中のマクロを停止します。


既知の問題
-------------------------------------------------
  - メニュー画面を表示させたまま液晶画面が自動で消灯する程度の間放置すると、
    メニュー終了時にゲーム画面が一瞬映ってすぐに暗転し、そのままになる。
    これは液晶の電源が切れている状態になっているだけなので、
    輝度変更ボタンを押すと画面が復帰する。

  - データのロード中にメニューを起動するとフリーズする割合が高い。
    新しいエリアに入ったときのロードや、ムービーの再生前のロードなど、
    何かをロードしている最中にメニューを呼び出さないように注意。

  - 英語がダメすぎ。
    いかんともしがたい。

最後に
-------------------------------------------------
  つい軽く見がちですが、プラグインは公式の仕組みではないし、
  PSP上で動作するゲームはプラグインの存在を考慮していません。
  従って、ゲームは常にフリーズする危険と隣り合わせにあることを忘れないようにしましょう。
  セーブが可能なゲームについてはこまめなセーブを行いましょう。
