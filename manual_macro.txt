MacroFire - 1.1.3 Version


Macro settings
-------------------------------------------------
  記録したボタン操作を繰り返し実行することを可能にします。

  Run once
    記録しているマクロを一度だけ実行する。

  Run infinity
    記録しているマクロを後述のStop macroを実行するまで
    延々と実行し続ける。

  Stop macro
    現在実行中のマクロを停止する。

  Start recording
    マクロを記録する。

  Stop recording
    マクロの記録を終了する。

  Recording analog stick (ON/OFF)
    マクロの記録において、アナログスティックの動作を記録するかを切り替える。
    マクロ記録中でも設定を変更可能。

  Edit macro
    記録しているマクロを編集する。
    操作や、マクロコマンドタイプについては後述のEdit macroの項目を参照。

  Clear macro
    記録しているマクロをクリアして、メモリを解放する。

  Create macro
    編集可能なマクロの最小限構成を作成する。
    具体的には、内容が Delay 0 ms のみのマクロを作成する。

  Load from MemoryStick
    メモリースティックからマクロファイルをロードする。
    (特定の状態でこれを選択すると操作不能に陥ることがあるので、必ずreadme.txtの既知の問題を参照のこと)
    

  Save to MemoryStick
    メモリースティックへ、現在保持しているマクロを保存する。
    (特定の状態でこれを選択すると操作不能に陥ることがあるので、必ずreadme.txtの既知の問題を参照のこと)


Edit macro
-------------------------------------------------
  Start recordingで記録したメインメモリ上のマクロを編集します。
  一行一行が実行すべきコマンドで、種類は以下のものが存在します。

    Delay: 一時停止する。
           直前のボタン状態を指定時間維持する。
           単位はms(ミリ秒)で1秒は1,000ミリ秒と等価。
           最大値は999,999,999ms(277時間46分39.999秒)になる。

    Buttons press: ボタンを押下する。
                   直前のボタン状態に追加される形でボタンを押す。
                     Buttons press: ×
                     Buttons press: ○
                   となっていると、最終的に○と×の二つが押される状態になる。

    Buttons release: ボタンを放す。
                      Buttons press同様に、直前の状態から指定されたボタンが放される形になる。
                        Buttons press  : ○ △
                        Buttons press  : ×
                        Buttons release: △
                      となっていると、最終的に○と×の二つが押される状態になる。

    Buttons change: ボタンの状態を変更する。
                    Buttons pressとButtons releaseが同時に起きたときに発生するコマンドで、
                    直前のボタン状態に関わらずボタン状態を完全に変更する。
                      Buttons press  : ○ △
                      Buttons press  : ×
                      Buttons release: △
                      Buttons change : □
                    となっていると、最終的に□ボタンのみが押される状態になる。

    Analog neutral: アナログスティックをニュートラルにする。
                    Analog moveで移動していたアナログスティックをニュートラルへ戻す。

    Analog move: アナログスティックを傾ける。
                 左上が(x,y)=(0,0)として、アナログスティックを任意の位置へ移動させる。

  エディットではこれらコマンドの値の変更と、
  これらコマンドそのものを挿入/削除を行うことができます。

  また、マクロを全てDeleteで削除しようとしても、完全には消せません。
  具体的には必ず Delay 0 ms が残ります。
  マクロを完全に消去するには、マクロのメインメニューからClear macroを実行してください。


注意
-------------------------------------------------
  Run once/Run inifinityによって実行中に、MacroFire EngineをOFFにしてゲームに戻ると、
  マクロはコマンド実行位置を先頭にリセットして、再実行を待ちます。
  つまり、MacroFir eEngineをOFFにしてゲームに戻ったあと、再びONにするとマクロは先頭から再実行されます。
  MacroFire Toggle Buttonsと組み合わせると、ゲーム上で擬似的にマクロをボタンで実行/停止できます。

  POPS(PS1エミュレータ)では、高速すぎる入力は無視されることがあります。
  Delayなしで次々にコマンドを実行するようなマクロをつくってもうまく動かないかもしれません。
  Delayの調整には注意してください。


v1.0.0で保存したマクロファイルが読み込めない問題について
-------------------------------------------------
  v1.0.0で保存したマクロファイルにはシグネチャがないため、
  それ以降のMacroFireで読み込むことができません。
  これを解決する方法は簡単です。

  まず、Windowsなどでマクロファイルをテキストエディタで開きます。
  そして先頭に以下の行を挿入するだけです。

    MACROFIRE-MACRO 1

  例えば、マクロファイルを開いたとき以下のようになっていたとします。

    ButtonsPress	2000
    Delay	10
	ButtonsRelease	2000

  これを以下のようにします。

    MACROFIRE-MACRO 1
    ButtonsPress	2000
    Delay	10
    ButtonsRelease	2000

  この先頭の行がシグネチャになります。
  この行で、このファイルがMacroFireに関係のあるファイルかを確認します。
