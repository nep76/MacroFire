MacroFire 2.3.1


Macro settings
-------------------------------------------------
  記録したボタン操作を繰り返し実行することを可能にします。

  Macro slot
    マクロのスロット番号。
    スロットごとにマクロは保存され、最大10まで保存可能。
    ただし、メモリ自体が足りない場合は、10全てを満足に使えるとは限らない。

  Buttons to run the macro
    ゲーム中にメニューを表示させずにマクロを実行するボタン。
    マクロ実行中にこのボタンを押すと逆に停止させることができる。

  Run once
    記録しているマクロを一度だけ実行する。

  Run infinity
    記録しているマクロを後述のStop macroを実行するまで
    延々と実行し続ける。

  Stop macro
    現在実行中のマクロを停止する。

  Start recording
    マクロを記録する。

  Stop recording
    マクロの記録を終了する。

  Recording analog stick (ON/OFF)
    マクロの記録において、アナログスティックの動作を記録するかを切り替える。
    マクロ記録中でも設定を変更可能。

  Edit macro
    記録しているマクロを編集する。
    操作や、マクロコマンドタイプについては後述のEdit macroの項目を参照。

  Clear macro
    記録しているマクロをクリアして、メモリを解放する。

  Load from MemoryStick
    メモリースティックからマクロファイルをロードする。
    (特定の状態でこれを選択すると操作不能に陥ることがあるので、必ずreadme.txtの既知の問題を参照のこと)

  Save to MemoryStick
    メモリースティックへ、現在保持しているマクロを保存する。
    (特定の状態でこれを選択すると操作不能に陥ることがあるので、必ずreadme.txtの既知の問題を参照のこと)


Edit macro
-------------------------------------------------
  Start recordingで記録したメインメモリ上のマクロを編集します。
  一行一行が実行すべきコマンドで、種類は以下のものが存在します。

    Delay: 一時停止する。
           直前のボタン状態を指定時間維持する。
           単位はms(ミリ秒)で1秒は1,000ミリ秒と等価。
           最大値は999,999,999ms(277時間46分39.999秒)になる。

    Buttons press: ボタンを押下する。
                   直前のボタン状態に追加される形でボタンを押す。
                     Buttons press: ×
                     Buttons press: ○
                   となっていると、最終的に○と×の二つが押される状態になる。

    Buttons release: ボタンを放す。
                      Buttons press同様に、直前の状態から指定されたボタンが放される形になる。
                        Buttons press  : ○ △
                        Buttons press  : ×
                        Buttons release: △
                      となっていると、最終的に○と×の二つが押される状態になる。

    Buttons change: ボタンの状態を変更する。
                    Buttons pressとButtons releaseが同時に起きたときに発生するコマンドで、
                    直前のボタン状態に関わらずボタン状態を完全に変更する。
                      Buttons press  : ○ △
                      Buttons press  : ×
                      Buttons release: △
                      Buttons change : □
                    となっていると、最終的に□ボタンのみが押される状態になる。

    Analog move: アナログスティックを傾ける。
                 左上が(x,y)=(0,0)として、アナログスティックを任意の位置へ移動させる。

    Rapidfire start: 連射を開始する。
                     Buttons pressと同じように連射を行うボタンを設定する。
                     PD/RDはPress delay/Release delayのことで、詳細はmanual_rapidfire.txtを参照。

    Rapidfire stop: 連射を停止する。


  エディットではこれらコマンドの値の変更と、
  これらコマンドそのものを挿入/削除を行うことができます。

  また、マクロを全てDeleteで削除しようとしても、完全には消せません。
  具体的には必ず Delay 0 ms が残ります。
  マクロを完全に消去するには、マクロのメインメニューからClear macroを実行してください。


注意
-------------------------------------------------
  POPS(PS1エミュレータ)では、高速すぎる入力は無視されることがあります。
  Delayなしで次々にコマンドを実行するようなマクロを作ってもうまく動かないかもしれません。
